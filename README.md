# Plotly Datasets Dashboard

A Streamlit dashboard showcasing Plotly's built-in datasets with interactive filtering and visualization.

## Features

- **Tips Dashboard**: Analyze restaurant tipping data with filters by day, time, and sex
- **Iris Dashboard**: Explore iris flower measurements with species filtering
- **Gapminder Dashboard**: Visualize global life expectancy and GDP data by year and continent

## Installation

```bash
pip install -r requirements.txt
```

## Running the App

```bash
streamlit run app.py
```

The app will open at `http://localhost:8501`

## Datasets

- **Tips**: Restaurant billing and tipping data
- **Iris**: Classic iris flower dataset
- **Gapminder**: Global development indicators

## Technologies

- Streamlit
- Plotly Express
- Pandas
